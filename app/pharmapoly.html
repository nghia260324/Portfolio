Ứng dụng mua thuốc trực tuyến PharmaPoly

Ứng dụng mua thuốc trực tuyến PharmaPoly

Đây là dự án tốt nghiệp với mục tiêu xây dựng một ứng dụng di động hỗ trợ người dùng đặt mua thuốc trực tuyến, quản lý đơn hàng và thanh toán qua VietQR hoặc COD.

Công nghệ và công cụ sử dụng trong dự án

Dưới đây là các công nghệ và công cụ tôi đã sử dụng để phát triển ứng dụng mua thuốc trực tuyến này, bao gồm từ phát triển frontend, backend, quản lý dự án đến tích hợp thanh toán.

IDE & Trình soạn thảo

  • Android Studio
  • Visual Studio Code

Quản lý mã nguồn & Quản lý dự án

  • GitHub
  • Trello
  • Figma
  • Draw.io

Backend & Database

  • Node.js
  • Express.js
  • MongoDB
  • JWT (JSON Web Token) - xác thực người dùng
  • Firebase:
    • Realtime Database
    • Authentication
    • SMS Authentication
    • Cloud Storage
    • Cloud Messaging (FCM) - gửi thông báo
  • Mongoose

Frontend & Template Engine

  • JavaScript
  • Handlebars (hbs)
  • HTML & CSS
  • Bootstrap

Thanh toán & Tích hợp

  • Casso API
  • VietQR
  • Webhook xử lý thanh toán realtime
  • Giao Hàng Nhanh (GHN):
    • Tạo đơn hàng qua API
    • Cấu hình webhook cập nhật trạng thái đơn hàng tự động

Công cụ hỗ trợ khác

  • Google Apps Script (xử lý Excel/Google Sheets tự động)
  • Postman (test API)
  • Render (build & deploy server)
  • Nodemon (tự động restart server)
  • Multer (xử lý upload file)
